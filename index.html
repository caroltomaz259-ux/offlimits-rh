⚙ Configuração do SharePoint

Conecte o sistema com a lista do SharePoint para sincronizar contratos e receber respostas automaticamente.
📋 Como encontrar essas informações:
1. Abra seu SharePoint no navegador
2. A URL será algo como: https://suaempresa.sharepoint.com/sites/rh
3. O nome do site é a parte depois de /sites/ (ex: rh)
4. O nome da lista você vê na página do SharePoint (ex: Contratos - Off-limits)
Exemplo: https://contoso.sharepoint.com
A parte depois de /sites/ na URL. Ex: se a URL é .../sites/rh, coloque: rh
O nome que aparece no SharePoint. Exatamente como está escrito.
Se o Power Automate salva as respostas em uma lista separada. Deixe em branco se não tiver.
🔐 Autenticação: Este sistema usa o login da Microsoft que já está ativo no seu navegador (cookies da sessão do SharePoint). Não é necessário configurar chaves de API separadas — funciona enquanto você estiver logada na sua conta Microsoft.
